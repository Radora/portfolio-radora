Real Estate DB model

A simple working database model of a property-managment system which I am currently working on

The business rules are:

  • There can be multiple Properties
    • Each Property can have multiple Buildings

  • Each Building can have a Manager
    • Each Building can have multiple rental Units and commercial space
    • Each Building will have Utilities
    • Each Building may have Features

  • Each Unit will have to be Inspected when a new Guest moves in
    • Each Unit can have multiple Guests
    • Units pricing may change
    • Each Unit has Features

  • Each Guest may have a car
  • Some Guests may be companies
